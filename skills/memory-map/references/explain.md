# Explain a Memory Map

## Goal

Explain what the existing memory map is doing and why the structure helps memory.

## What to Cover

- the central concept
- why the novelty or contrast matters
- how association nodes connect the idea to other notes
- how analogy or elaboration makes the concept easier to recall
- how the chunks reduce cognitive load

## Tone

- concise
- practical
- focused on memory benefit, not just visual layout
