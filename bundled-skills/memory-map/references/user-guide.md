# Memory Map User Guide

Memory Map is the visual understanding module in EngramQuest.

It creates an Obsidian Canvas file that helps users remember difficult concepts through:

- contrast
- association
- analogy
- contextual anchoring
- chunking

## Example prompts

- `Create a memory-map for [note].md`
- `Update the memory-map for [note].md`
- `Explain this memory-map`

## Output

The generated file should be saved as `<source-note-name>-memory.canvas`.
