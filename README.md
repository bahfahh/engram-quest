## English

**The first AI-native learning system for Obsidian.**

![EngramQuest Exhibit](assets/exhibit.png)

EngramQuest is more than a plugin; it's an ecosystem that transforms static notes into "Dynamic Learning Paths." By installing **AI Skills** within Obsidian, your AI tools (Claude Code, Gemini CLI, Cursor) will learn how to craft interactive maps and memory cards tailored specifically for you.

### ⚡ Quick Start

Master any topic in four simple steps:

1.  **Install Plugin:** Install **EngramQuest** via the Obsidian Community Plugins or download the latest version from [GitHub Releases](https://github.com/bahfahh/engram-quest/releases).
2.  **Install Skills:** Go to `Settings -> EngramQuest -> AI Skills` and click **Install** for your preferred tool (e.g., Claude or Cursor).
    ![Install Skills](assets/install-skills.png)
3.  **Generate Content:** In your AI tool, type: *"Turn `Note Name.md` into a **quest-map**"* or *"Build a **review deck** about Python."*
4.  **Start Learning:** Click the **EngramQuest icon** in the Obsidian ribbon to open the Hub, switch to the relevant tab, and start your challenge!

### ✨ Key Features

#### 🗺️ Quest Map
Turn long, dry notes into game-like "Island Stages."
- **Interactive Challenges:** Includes multiple-choice, ordering, matching, and more.
- **Difficulty Scaling:** Request **Easy**, **Medium**, or **Hard** modes from your AI.
- **Visual Progress:** Track mastery for each chapter and unlock the final Boss challenge.

#### 🧠 Memory Map
Visualize abstract concepts using Obsidian Canvas.
- **Association Building:** AI analyzes note relationships to create visual knowledge "chunks."
- **Deep Intuition:** Master difficult topics through contrast, analogy, and contextual anchoring.

#### 🃏 Review Deck
Scientific long-term memory powered by the **FSRS Algorithm**.
![Review Deck Demo](assets/review-deck-demo.png)
- **Smart Scanning:** Automatically detects `question :: answer` cards tagged with `#flashcards/`.
- **Triple-level Hints:** Stuck? AI provides L1 (Clue), L2 (Vault Context), or L3 (Direct Hint).
- **Legacy Mode:** Optional support for scanning plain `::` cards without tags for easy migration.

### 🔬 Why It Works
EngramQuest is built on three pillars of cognitive science:
1.  **Spaced Repetition:** Maximizes memory efficiency using the FSRS algorithm.
2.  **Retrieval Practice:** Forced active recall is far more effective than passive re-reading.
3.  **Elaborative Encoding:** Translates text into visual structures for deeper cognitive links.

### ❓ FAQ
**Q: Do I have to use AI?**  
**A:** Highly recommended. While you can manually write flashcards, the core value of **Quest Map** and **Memory Map** lies in AI automation. You can use the free **Gemini CLI** or **Claude Code** to generate high-quality content quickly.

**Q: Where is my progress stored?**  
**A:** All data is stored as standard Markdown comments (SR comments) directly in your notes. You own your data forever.

**Obsidian 第一款 AI 原生學習系統。**

![EngramQuest 展示](assets/exhibit.png)

EngramQuest 不只是外掛，它是一個將靜態筆記轉化為「動態學習路徑」的生態系。透過在 Obsidian 內部安裝 **AI Skills**，你的 AI 工具（Claude Code, Gemini CLI, Cursor）將學會如何為你量身打造互動地圖與記憶卡片。

### ⚡ 快速上手 (Quick Start)

只需要簡單四步，即可開啟你的 AI 學習之旅：

1.  **安裝外掛：** 在 Obsidian 安裝 **EngramQuest** 並啟用（可透過社群外掛商店或從 [GitHub Releases](https://github.com/bahfahh/engram-quest/releases) 下載最新版本）。
2.  **安裝 Skills：** 前往 `設定 -> EngramQuest -> AI Skills`，點擊你常用的工具（如 Claude 或 Cursor）進行 **Install**。
    ![安裝 Skills](assets/install-skills.png)
3.  **生成內容：** 在你的 AI 工具中輸入：*"幫我把 `筆記名稱.md` 做成一個 **quest-map**"* 或 *"幫我建立關於 Python 的 **review deck**"*。
4.  **開始學習：** 點擊 Obsidian 左側邊欄的 **EngramQuest 圖示** 打開 Hub，選擇對應分頁開始挑戰！

### ✨ 核心特色

#### 🗺️ Quest Map (學習地圖)
將枯燥的長篇筆記變成像遊戲一樣的「關卡島嶼」。
- **關卡挑戰：** 包含選擇題、排序題、連連看等多種互動形式。
- **難度分級：** 可要求 AI 生成 **Easy** (初探)、**Medium** (鞏固) 或 **Hard** (精通) 模式。
- **視覺進度：** 直觀看到每個章節的掌握程度，解鎖最終的 Boss 挑戰。

#### 🧠 Memory Map (視覺記憶)
利用 Obsidian Canvas (畫布) 將抽象概念具象化。
- **關聯建構：** AI 會自動分析筆記間的關聯，建立視覺化的知識區塊 (Chunking)。
- **深度理解：** 透過對比、類比與情境錨定，幫你攻克最難記的知識點。

#### 🃏 Review Deck (長期記憶)
基於科學的 **FSRS 演算法**，在「快要忘記」的黃金時間提醒你複習。
![複習卡片展示](assets/review-deck-demo.png)
- **精準複習：** 預設掃描帶有 `#flashcards/` 標籤的 `問題 :: 答案` 筆記。
- **三段式提示：** 當你想不起來時，AI 會提供 L1 (線索)、L2 (筆記關聯) 或 L3 (直接提示)。
- **Legacy 模式：** 支援開啟設定以掃描舊有的純 `::` 格式筆記，輕鬆遷移。

### 🔬 為什麼有效？
EngramQuest 結合了三大學習科學原理：
1.  **間隔重複 (Spaced Repetition):** 使用 FSRS 演算法最大化長期記憶效率。
2.  **提取練習 (Retrieval Practice):** 強迫大腦「先回想，再看答案」，效果遠勝重複閱讀。
3.  **精細編碼 (Elaborative Encoding):** 將文字轉化為視覺結構，建立更深層的認知連結。

### ❓ 常見問題
**Q: 我一定要用 AI 嗎？**  
**A:** 強烈建議使用。雖然可以手動撰寫複習卡片，但 **Quest Map** 與 **Memory Map** 的自動化生成是本外掛的核心價值。你可以搭配免費的 **Gemini CLI** 或 **Claude Code** 來快速建立高品質的學習內容。

**Q: 我的學習進度存哪裡？**  
**A:** 所有的複習數據都以標準 Markdown 註釋形式存放在你的筆記中，資料永遠屬於你，不依賴任何雲端服務。
